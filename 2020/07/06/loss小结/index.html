Loss小结

Hinge LOSS

一般的f(x)是预测值，在-1到1之间， y是目标值(-1或1)。其含义是，f(x)的值在-1和+1之间就可以了，并不鼓励|f(x)|>1 ，即并不鼓励分类器过度自信，让某个正确分类的样本距离分割线超过1并不会有任何奖励，从而使分类器可以更专注于整体的误差。

L1（MAE） L2(MSE)

优点：各点都连续光滑，方便求导，具有较为稳定的解

缺点：不是特别的稳健，为什么？因为当函数的输入值距离中心值较远的时候，使用梯度下降法求解的时候梯度很大，可能导致梯度爆炸

优点：无论对于什么样的输入值，都有着稳定的梯度，不会导致梯度爆炸问题，具有较为稳健性的解

缺点：在中心点是折点，不能求导，不方便求解

focal loss

focal loss: a 平衡正负样本重要性，focal loss能区分easy/Hard样本 a=0.25也就是让正负样本重要性比例为1:3

GHM

GHM：改进focal loss，类别的不平衡可以归结为难易样本的不平衡，难易样本的不平衡可以归结为梯度的不平衡
其中，如果让模型过多关注难分样本会引发一些问题，比如样本中的outliers，总而言之，我们不能过多关注易分样本，和难分样本，我们通过梯度函数来抑制简单样本和困难样本，也就是用二分类交叉熵和梯度密度函数的倒数相乘即可

smooth L1

Fast RCNN中提出

L2损失函数在训练初期，导数大，导致训练不稳定
L1损失函数在训练后期，损失函数在0处附近波动，很难收敛到更高的精度
smooth L1避免了L1L2的缺点

iou loss

认为smooth L1中独立的回归四个点这种做法假设了这4个点是独立的，但是他们是有相关性的

实际评价标准是IOU，多个检测框可能有相同的smooth L1 loss，但IOU相差很大

提出IOU LOSS

GIOU loss

  1. 两个框没有相交，无法反应两者距离的大小，IOU=0，无法优化，失去可导的性质
  2. IoU无法精确的反映两者的重合度大小

约束为框A和框B的最小外接矩形

DIOU LOSS & CIOU LOSS

当目标框完全包裹预测框的时候，IoU和GIoU的值都一样，此时GIoU退化为IoU, 无法区分其相对位置关系

好的目标框回归损失应该考虑三个重要的几何因素：重叠面积，中心点距离，长宽比。

DIOU LOSS只考虑了重叠面积和中心点距离

DIOU是考虑了框A和框B的最小外接矩形的对角线距离

CIOU LOSS三个都考虑了，在DIOU的基础上加了对于宽高的约束
